What is CSS ?

  1. CSS Stand for Cascading Stylesheet
  2. CSS describe how HTML, element are to diplayed on screen, paper or other media
  3. CSS is not neccssary CSS save a lot of work.It can controll the layout of multiple web pages all at once
  4. External stylesheet are stored in CSS file

CSS can be added in HTML in three ways:

  • Inline- by usinfg the style attributes inside HTML elements
  • Internal- by using a style element in the head section
  • External- by using a link element to link an external CSS file
  • The most common way to add CSS is to keep the style in external CSS file. How ever, in the tutorial we will use inline and internal style, because this is eaiser to demonstrate, and eaiser for you try it yourself.

When comparing CSS class vs ID,the differnce is that CSS class applies a style to multiple elements.ID, on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL.to link directly to an element and it's used by JavaSript.

In CSS, selectors are used to target a specific element or range of elements on a webpage.Once an element has been targeted, a style or set of styles can be applied to the elements

There is a wide range of selectors avialable.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
